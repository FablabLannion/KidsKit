# Arduino Kit Joystick/Servo
* 1 joystick
* 1 servomotor

Control servo movement with the joystick.

## BOM
* 1 arduino nano
* 1 "ps2" joystick for arduino
* 1 microservo 9g
* 6 M-M breadboard wires
* 3 F-F breadboard wires
* 1 laser cuted plate (hardware/Kit_joyservo.dxf)
* 1 mini-breadboard
* 2 10mm M4 bolts
* 4 20mm M4 bolts
